# AddressableAutomation
* 모든 파일에 대한 어드레서블 라벨을 폴더별로 그룹화하는 매니저입니다.

# 특징
* 특정 폴더(AllResources) 내부에 존재하는 모든 파일들을 자동으로 어드레서블에 등록합니다.
* 어드레서블에 등록되는 라벨의 이름은 폴더의 Path로 설정됩니다.
